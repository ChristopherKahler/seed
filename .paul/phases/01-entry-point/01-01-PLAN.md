<objective>
## Goal
Create the Skillsmith-compliant entry point file (seed.md) with correct YAML frontmatter, all 5 XML sections (activation, persona, commands, routing, greeting), and proper routing to tasks and data layer.

## Purpose
The entry point is the identity and routing layer for the entire skill. Without it, nothing loads. It must be thin (identity + routing only) — all process logic lives in tasks/.

## Output
- `seed.md` — complete, Skillsmith-compliant entry point
</objective>
